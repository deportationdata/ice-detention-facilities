NOCCC (Northern California Collaborative for Immigrant Justice) data

NOCCC_holdroom_research.csv  — hand-researched list of ICE hold rooms
  with detention facility codes, names, and addresses. Compiled by NOCCC.

holdroom_office_mapping.csv  — crosswalk from ICE hold-room detention
  facility codes to ICE field office UUIDs used by the coldCounter
  project. See https://github.com/CovenAnalytica/coldCounter for the
  UUID generation formula.

ice-offices.xlsx             — ICE offices reference table (names,
  addresses, coordinates) from the Deportation Data Project
  ice-offices repository at
  https://github.com/deportationdata/ice-offices
  Retained to capture correct version
